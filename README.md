# Pokémon App 

Приложение на React для отображения карточек покемонов с использованием данных из PokéAPI.

## Demo
Live version: https://annarats.github.io/pokemon/

## Project Description

Приложение позволяет:

- загружать данные о покемонах с внешнего API (PokéAPI)  
- отображать карточки покемонов в сетке  
- поиск по персонажам и открытие отдельной страницы с информацией о покемоне
- добавление к каждому покемону ссылки на внешний ресурс
  
### Работа с API

- Получение данных через **REST API** (PokéAPI)  
- Обработка полученных данных для отображения на UI
  
## Tech Stack
- React
- JavaScript (ES6)
- CSS
- PokéAPI
